# Modern Data Warehouse Project (SQL Server)

This project demonstrates the design and implementation of a modern data warehouse using Microsoft SQL Server.

***Project Overview**

* Data architecture: The data architecture for this project follows Medallion Architecture Bronze, Silver, and Gold layers
  *Bronze Layer: Stores raw data from source systems (CRM & ERP)
  *Silver Layer: Cleans, standardizes, and transforms the data
  *Gold Layer: Provides business-ready data for analytics and reporting
* ETL Pipelines : Extracting, transforming, and loading data from source systems into the warehouse
* Data Modeling: Developing fact and dimension tables
